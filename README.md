# SpaceX Launch Site Interactive Visualization

## Project Overview

This project uses interactive geospatial visualization to analyze SpaceX launch site locations and launch outcomes.

The goal is to explore whether launch site geography, proximity to the coast, and surrounding infrastructure may be related to launch success patterns.

## Business and Analytical Context

Launch site selection is an important part of aerospace operations. A good launch site must consider geography, safety, cost, infrastructure, and trajectory requirements.

This project demonstrates how interactive maps can help analysts visually explore launch sites and identify geographical patterns.

## Objectives

The project focuses on three main tasks:

1. Mark all SpaceX launch sites on an interactive map
2. Visualize successful and failed launches by location
3. Analyze launch site proximity to coastlines, highways, railways, and other geographical features

## Dataset

The project uses SpaceX launch site location data, including:

- Launch site name
- Latitude
- Longitude
- Launch outcome class

## Methods Used

- Data loading with Pandas
- Geographical coordinate analysis
- Interactive map generation
- Marker clustering
- Launch outcome visualization
- Distance and proximity exploration
